# Nutri track API

A REST API for managing user profiles, meals, meal plans, dashboards, and
administration. Built with Express, TypeScript, Prisma, and Supabase.

## Getting started

Install the dependencies:

```bash
yarn install
```

Create an environment file and configure its values:

```bash
cp .env.example .env
```

Start the development server:

```bash
yarn dev
```

## Build

```bash
yarn build
```
